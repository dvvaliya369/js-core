Component Usage

This button component supports the following properties:

  • text - The button text (default: "Click Me")
  • type - Button style: primary, secondary, success, danger, warning, info (default: primary)
  • disabled - Boolean to disable the button
  • on-click - Function to call when button is clicked

Example Usage:

<custom-button 
    text="Save" 
    type="success" 
    disabled="false"
    on-click="myFunction()">
</custom-button>
