Button Component Demo

This demo showcases the Button component with various configurations using plain HTML and CSS classes.

Basic Variants

<button class="btn btn--primary btn--md">Primary</button>
<button class="btn btn--secondary btn--md">Secondary</button>
<button class="btn btn--outline btn--md">Outline</button>
<button class="btn btn--ghost btn--md">Ghost</button>
<button class="btn btn--danger btn--md">Danger</button>

Size Variants

<button class="btn btn--primary btn--sm">Small</button>
<button class="btn btn--primary btn--md">Medium</button>
<button class="btn btn--primary btn--lg">Large</button>

States

<button class="btn btn--primary btn--md">Normal</button>
<button class="btn btn--primary btn--md btn--disabled" disabled>Disabled</button>
<button class="btn btn--primary btn--md btn--loading">...</button>

With Icons

<button class="btn btn--primary btn--md">
  <span class="btn__icon btn__icon--start">[icon]</span>
  <span class="btn__content">Download</span>
</button>

Full Width



<button class="btn btn--primary btn--md btn--full-width">Full Width Button</button>
