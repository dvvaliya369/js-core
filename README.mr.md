# <span style="color:red">JS कोर युटिलिटीज</span>

सामान्य प्रोग्रामिंग कार्यांसाठी JavaScript युटिलिटी फंक्शन्सचा संग्रह.

## वैशिष्ट्ये

### डिबाउन्स फंक्शन

`debounce` फंक्शन एखाद्या फंक्शनचे अंमलबजावणी विलंबित करते जोपर्यंत शेवटच्या वेळी ते आमंत्रित केल्यापासून निर्दिष्ट प्रतीक्षा वेळ उलटून जात नाही. हे फंक्शन किती वेळा चालू शकते हे मर्यादित करण्यासाठी उपयुक्त आहे, जसे की शोध इनपुट किंवा रीसाइझ इव्हेंट्समध्ये.

#### वापर

```javascript
const { debounce } = require('./utils.js');

// मूलभूत वापर
const debouncedSave = debounce(saveData, 300);
debouncedSave(); // पुढील कॉल्स नसल्यास 300ms नंतर अंमलात येईल

// तात्काळ अंमलबजावणी
const debouncedImmediate = debounce(saveData, 300, true);
debouncedImmediate(); // ताबडतोब अंमलात येते, नंतर पुढील कॉल्स डिबाउन्स करते
```

#### पॅरामीटर्स

- `func` (function): डिबाउन्स करण्यासाठी फंक्शन
- `wait` (number): विलंब करण्यासाठी मिलिसेकंदांची संख्या
- `immediate` (boolean, वैकल्पिक): जर खरे असेल तर, ट्रेलिंग एजऐवजी लीडिंग एजवर फंक्शन ट्रिगर करा

#### परतावा

इनपुट फंक्शनची डिबाउन्स केलेली आवृत्ती.

## स्थापना

रिपॉझिटरी क्लोन करा आणि तुमच्या प्रोजेक्टमध्ये `utils.js` समाविष्ट करा.

## चाचणी

डिबाउन्स फंक्शन सत्यापित करण्यासाठी चाचणी फाइल चालवा:

```bash
node test_utils.js
```
